## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/owagajadoh/portfolio.git
   cd portfolio
   ```

2. **Open in browser**
   ```bash
   open index.html
   ```

3. **Enable GitHub Pages** (Optional)
   - Go to Settings → Pages
   - Select `main` branch
   - Your site will be live at `https://owagajadoh.github.io/portfolio`

## 📝 Customization

### Update Contact Information

Edit these sections in `index.html`:

```html
<!-- Email -->
<a href="mailto:your-email@example.com">Your Email</a>

<!-- WhatsApp -->
<a href="https://wa.me/yourphonenumber">Your Phone</a>

<!-- Location -->
<span>📍 Your City, Country</span>
```

### Modify Services

Edit the services grid in `index.html` to match your offerings.

### Add Real Projects

Replace placeholder project cards with actual work:

```html
<div class="project-card">
  <div class="project-image" style="background:url('your-project-image.jpg')"></div>
  <div class="project-content">
    <h3>Your Project Title</h3>
    <p>Project description...</p>
    <div class="project-stack">
      <span>Tech 1</span><span>Tech 2</span>
    </div>
  </div>
</div>
```

### Update Pricing

Edit pricing section with your rates and engagement models.
